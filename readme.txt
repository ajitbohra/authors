== Installation ==

***Manual Installation***
 1. [Download](https://wordpress.org/plugins/authors/) "authors".
 2. Upload the `authors` directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
 3. Activate `authors` from your Plugins page.
